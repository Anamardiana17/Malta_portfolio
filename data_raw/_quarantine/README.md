# _quarantine

This folder contains metadata and citation guidance for HTML snapshot source materials used as contextual support in Malta_portfolio.

## Purpose
These materials are retained as review snapshots for source-traceability and controlled public-repo treatment.

## Public repo treatment
This public repo prefers:
- metadata manifests
- bibliographic references
- official links
- citation guidance

instead of re-hosting third-party HTML snapshot files where licence clarity is limited.

## Files
- `_quarantine_source_manifest.csv` — metadata register for HTML snapshot source materials in this folder

## Important note
These materials are used as contextual support only and should not be overstated as direct spa operating truth.
