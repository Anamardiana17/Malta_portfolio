# Malta International Airport (MIA) source folder

This folder contains metadata and citation guidance for Malta International Airport-related source materials used as contextual support in Malta_portfolio.

## Purpose
These materials are used for:
- airport traffic context
- annual passenger movement summaries
- tourism-access and mobility framing

## Public repo treatment
This public repo prefers:
- metadata manifests
- bibliographic references
- official links
- citation guidance

instead of re-hosting third-party PDF files where licence clarity is limited.

## Files
- `mia_source_manifest.csv` — metadata register for MIA report files previously used as source support

## Preferred citation
`Source: Malta International Airport plc, Annual Summary Report [year].`

## Important note
MIA materials are used as contextual support only and should not be overstated as direct spa operating truth.
