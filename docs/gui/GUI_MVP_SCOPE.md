# GUI MVP Scope

Purpose:
A KPI-Governed Operations Control Panel layered on top of Malta_portfolio outputs.

Guardrails:
- GUI is a control and presentation layer only
- no synthetic intra-day logic
- no unsupported staffing inference
- no replacement of core repo pipeline
- recruiter-facing and management-facing usability
- preserve methodological defensibility

MVP Panels:
1. Data Input Panel
2. Processing / QA Panel
3. KPI / Executive Dashboard Panel
4. Decision Support Panel
5. Export / Reviewer Pack Panel
