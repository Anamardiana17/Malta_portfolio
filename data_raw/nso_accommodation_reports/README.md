# nso_accommodation_reports

This folder contains metadata and citation guidance for source materials used as contextual support in Malta_portfolio.

## Purpose
These materials are used for:
- accommodation demand context
- tourism stay-volume support
- seasonal lodging reference framing

## Public repo treatment
This public repo prefers:
- metadata manifests
- bibliographic references
- official links
- citation guidance

instead of re-hosting third-party PDF files where licence clarity is limited.

## Files
- `nso_accommodation_reports_source_manifest.csv` — metadata register for PDF source materials in this folder

## Preferred citation
`Source: National Statistics Office (Malta), collective accommodation report / news release [year].`

## Important note
These materials are used as contextual support only and should not be overstated as direct spa operating truth.
