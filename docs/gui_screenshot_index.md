## Reviewer Framing

This GUI is positioned as a governed control layer above validated `Malta_portfolio` artifacts.
It does not replace the core pipeline, does not override portfolio methodology, and does not present external context or active analytical months as direct operational truth from raw upload events.

The screenshot sequence is intended to show this reviewer-facing flow:

`governed intake -> profiling -> manual review -> accepted gate -> controlled processing -> KPI visibility -> decision support -> reviewer-facing traceability`
