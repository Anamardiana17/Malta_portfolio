# nso_inbound_tourism

This folder contains metadata and citation guidance for HTML snapshot source materials used as contextual support in Malta_portfolio.

## Purpose
These materials are used for:
- inbound tourism context
- source-page traceability
- reviewer-facing citation support

## Public repo treatment
This public repo prefers:
- metadata manifests
- bibliographic references
- official links
- citation guidance

instead of re-hosting third-party HTML snapshot files where licence clarity is limited.

## Files
- `nso_inbound_tourism_source_manifest.csv` — metadata register for HTML snapshot source materials in this folder

## Preferred citation
`Source: National Statistics Office (Malta), inbound tourism page or release [year].`

## Important note
These materials are used as contextual support only and should not be overstated as direct spa operating truth.
