# GUI Control Panel

This GUI is a control and presentation layer on top of Malta_portfolio outputs.

Design principles:
- no replacement of core pipeline
- no synthetic intra-day logic
- no unsupported staffing inference
- recruiter-facing and management-facing usability
- preserve methodological defensibility

Current MVP scope:
- artifact discovery
- artifact readiness status
- panel routing
- executive dashboard preview
- decision support preview
- reviewer pack readiness
